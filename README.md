**Fashion Advice App**

Welcome to the Fashion Advice App! This application provides personalized outfit recommendations for various occasions, helping users dress appropriately and stylishly. Whether you're preparing for a casual brunch, a formal event, or a job interview, this app has you covered.

**Features**

Occasion-Based Outfit Suggestions: Get tailored fashion advice for specific events like weddings, job interviews, casual outings, and more.

Keyword Matching: The app uses a dataset of pre-defined inputs and outputs to match user queries with relevant fashion advice.

Easy to Use: Simply input your occasion, and the app will provide detailed outfit recommendations.

Scalable Dataset: The app uses a JSON dataset, making it easy to add new occasions and outfit suggestions.

**How It Works**

Dataset Structure: The app uses a JSON file to store outfit advice for various occasions. Each entry includes:

occasion: The event or scenario (e.g., "casual weekend brunch").

advice: Detailed outfit recommendations for the occasion.

keywords: A list of keywords associated with the occasion for matching user input.

Reverse Indexing: The app builds a reverse index to quickly match user input keywords with relevant advice.

User Interaction: Users input an occasion, and the app processes the input to retrieve matching outfit advice from the dataset.
